# Advanced Features Deep Dive

## Filters and saved views
Use / to focus the search bar and enter text, labels, or status queries.
Press f for the quick filter menu and Ctrl+S to save a filter. Saved
filters can be recalled with function keys.

## Column manager
Use the column manager to show, hide, or reorder columns in issue lists.
Alt+Left and Alt+Right move the focused column, and Alt+Shift+Left or
Alt+Shift+Right resize it.

## Dependency visualization
The Dependencies view renders issue relationships as trees. Use it to
confirm upstream blockers and validate dependency edits.

## Label intelligence
Labels support ranking, autocomplete, and dimension patterns like
state:patrol. The Labels view shows usage counts and assists cleanup.

## Bulk actions
Select multiple issues with Space, then use bulk actions to apply labels
or update fields across many issues at once.

## Performance overlay
Press Ctrl+P or F12 to toggle performance statistics while you work.

## Notifications and history
Use Shift+N to open notification history and review recent actions.
Open issue history from the issue detail view to see the timeline.
